Tarea de Kodigo sobre: Frotend react app usando Kodigo-api

Para realizar esta actividad se debe de consumir una rest api llamada Kodigo API. Usando React se debe de implementar lo siguiente:

 ·         Inicio de sesión de usuarios con JWT a nivel de cliente.

·         La aplicación debe de contar con las siguientes vistas:

o   Página de inicio: En esta página se debe de crear una landing page atractiva para los usuarios con la información de los bootcamps que la Kodigo API brinda.

o   Crear nuevo usuario:  Formulario de creación de nuevo usuario.

o   Inicio de sesión de usuario: Implementación de login de usuarios y manejo de JWT en localStora o Cookies.

o   Dashboard de usuarios: Pagina de detalles sobre los bootcamps que la Kodigo API brinda.

En esta actividad se deben de implementar las diferentes librerias de reactJs que nos ayuden a desarrollar las funcionalidades como el routing, manejo de formularios y de API’s; asi como también la correcta implementación de los React hooks. 

Como abrirlo.

Despues de descargar el proyecto abre visual studio code.
Luego Abre una terminal dentro de VS Code (Ctrl + ñ o Ctrl + Shift + ~) y ejecuta: npm install
Despues ejecuta: npm run dev. VS Code mostrará algo como: VITE v4.4.9  ready in 300 ms Local: http://localhost:5173/

Haz clic en el enlace o ábrelo en tu navegador:
http://localhost:5173/
